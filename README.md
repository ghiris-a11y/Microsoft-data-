# MSFT Equity Research Dashboard

This project performs financial ratio analysis and valuation of Microsoft Corporation (MSFT) using publicly available financial data.

## Features

- Fetches financial statements using `yfinance`
- Computes key financial ratios
- Performs DCF valuation
- Interactive dashboard built with Streamlit
- Modular Python code structure

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/msft-equity-research.git
   cd msft-equity-research
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the Streamlit app:
   ```bash
   streamlit run streamlit_app/app.py
   ```
